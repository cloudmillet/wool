# wool
Pylint Examples | Pylint Runner Script | Pylint Github Action


### Installation
```
pip install pylint
```


### Command Line Execution
```
pylint src/
```


### Simple Pylint Example
```
python simple.py
```


### Full Pylint Example
```
python lint.py
```


### Full Pylint Example with Adjustable Threshold and Path
```
python lint.py --path ./src/ --threshold 3 
```
